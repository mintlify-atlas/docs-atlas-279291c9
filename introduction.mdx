NdLinear is an open-source PyTorch module by [Ensemble AI](https://ensemblecore.ai) that replaces `nn.Linear` without collapsing your tensors into flat vectors. Instead of flattening multi-dimensional inputs, NdLinear applies a sequence of dimension-wise linear transformations — one per dimension — preserving the spatial and structural relationships in your data throughout the forward pass.

The approach is described in the paper [NdLinear Is All You Need for Representation Learning](https://arxiv.org/pdf/2503.17353).

## How it works

A standard `nn.Linear(in, out)` flattens the input and applies a single weight matrix of shape `(in, out)`. NdLinear instead accepts an `input_dims` tuple and a `hidden_size` tuple of the same length, and applies one `nn.Linear` per dimension via transpose, flatten, apply, and restore operations. The result is an output tensor with the same number of dimensions as the input, but with each dimension size transformed to the corresponding value in `hidden_size`.

```python
import torch
from ndlinear import NdLinear

# Input: batch of images with shape (batch, H, W, C)
layer = NdLinear(input_dims=(28, 28, 3), hidden_size=(64, 64, 6))
output = layer(torch.randn(32, 28, 28, 3))
# output.shape == (32, 64, 64, 6)
```

The `transform_outer=True` default processes dimensions from outermost to innermost. Each dimension's transformation is aware of the others through sequential composition.
