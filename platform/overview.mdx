The Ensemble Platform is a commercial service built on top of NdLinear. Where NdLinear gives you the primitive to manually replace `nn.Linear` layers in your own code, the platform automates the entire compression pipeline: it accepts your model, identifies which layers to replace, tunes the decomposition hyperparameters, and returns a smaller, faster model ready for deployment.

## How it works

<Steps>
  <Step title="Upload your model">
    Submit any PyTorch model through the platform. No code changes required.
  </Step>
  <Step title="Automatic compression">
    The platform swaps eligible layers for NdLinear equivalents and tunes decomposition hyperparameters to meet your target constraints.
  </Step>
  <Step title="Download the compressed model">
    Receive a smaller model with the same architecture interface, ready for fine-tuning or deployment.
  </Step>
</Steps>

## Relationship to NdLinear

NdLinear is the open-source PyTorch module that powers the compression. You can use it directly in your own code to manually decompose `nn.Linear` layers — see the [NdLinear documentation](/). The Ensemble Platform wraps that same primitive in an automated service, handling layer selection, hyperparameter search, and validation so you don't have to.

## Export options

Compressed models can be exported to several deployment formats, including ONNX, TensorRT, and SNPE. See [Model Export](/platform/export) for details.
