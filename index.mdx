NdLinear is a PyTorch module by [Ensemble AI](https://ensemblecore.ai) that replaces `nn.Linear` without any architectural changes. Instead of flattening your tensors, NdLinear transforms them across a structured set of vector spaces — preserving multi-dimensional structure and capturing dependencies that standard layers discard.

## Why NdLinear?

Standard `nn.Linear` flattens multi-dimensional tensors before applying a weight matrix, discarding the structural relationships between dimensions. NdLinear preserves that structure by applying a sequence of dimension-wise transformations — one linear layer per dimension — capturing cross-dimensional dependencies with far fewer parameters.

## Get started

<Steps>
  <Step title="Install NdLinear">
    ```bash
    pip install ndlinear
    ```
  </Step>
  <Step title="Import and replace your linear layer">
    ```python
    from ndlinear import NdLinear

    # Before: nn.Linear(784, 256)
    # After:
    layer = NdLinear(input_dims=(28, 28), hidden_size=(16, 16))
    ```
  </Step>
  <Step title="Run a forward pass">
    ```python
    import torch

    x = torch.randn(32, 28, 28)
    output = layer(x)  # shape: (32, 16, 16)
    ```
  </Step>
  <Step title="Explore guides and examples">
    See how to integrate NdLinear into [CNNs](/guides/cnn), [Transformers](/guides/transformer), [MLPs](/guides/mlp), and more.
  </Step>
</Steps>
